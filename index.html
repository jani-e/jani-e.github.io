How to make an unordered list with bullets:

  • Start with <ul> and </ul> tags.
  • Then inside it create bullets with <li>tag.
  • Don't forget the </ul> tag at the end of the list!

But what if I wanted to use numbers in my list?

  1. You can use the <ol> and </ol> tags.
  2. Then same as before: list things with <li> tag.

You can also make lists including descriptions.

Here you use a different tag called <dl>.
Use this to create lists that include defitions.
The next step is to add the tag <dt>.
Which dictates the item.
To add the description under it.
Use the tag <dd>.
